    <h2> 게임 설명 </h2>
    <h3> 스토리 </h3>
     당신은 혼자 남아 집을 지키고 있다. <br><br>
     누군가의 시선, 이상한 소리가 들린다. <br><br>
     옷장과 문을 확인해야 한다.
    <h3> 게임 플레이 </h3>
     A와 D키를 이용해 캐릭터를 앞뒤로 움직이고 <br><br>
     E키를 활용해 문, 옷장 등과 상호작용한다. <br><br>
     불이 켜진 상태에서 문, 옷장을 확인하고 책상으로 돌아온다.
